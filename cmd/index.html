Angular Commands

Kiểm tra phiên bản đã cài đặt:

  • Kiểm tra phiên bản node:
    node -v
  • Kiểm tra phiên bản npm:
    npm -v
  • Kiểm tra phiên bản angular:
    ng v

Cài đặt Typescript

npm install -g typescript

@angular/cli@9.1.12

npm install -g @angular/cli@9.1.12

Build web dạng thường

ng serve --open

Build web ở port khác (Mặc định port 4200)

ng serve --port 1234 --open

Build web theo ngôn ngữ

ng serve --configuration=production-vi --port 4200 --open
ng serve --configuration=production-en --port 4200 --open

Build web với fake host domain

ng serve --o --host "my-domain.vn" --port 80

Build file i18n messages.xlf

npm run xi18n

Build web

npm run build-i18n
