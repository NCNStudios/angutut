1.1. Cài đặt NodeJS + NPM

NodeJS

Phiên bản tối thiểu cần cài đặt: 8.11

Cài đặt phiên bản mới nhất tại: https://nodejs.org/en/ Quá trình cài đặt bình thường như những phần mềm khác. Tuy nhiên cần lưu ý chọn cài đặt đồng thời npm:

Kiểm tra phiên bản đã cài đặt:

  • Kiểm tra phiên bản node:
    node -v
  • Kiểm tra phiên bản npm:
    npm -v

3.2. Project Angular có sẵn

Có thể clone source từ nguồn chính thống (Gitlab) hoặc download file source nhờ vào mối quan hệ thân thiết mà xin chui được

Cấu trúc source gốc

 Không có thư mục node_modules

Cây thư mục và chức năng của ẻm
  • e2e: Thư mục chứa các file test... Không cần quan tâm
  • src: Tất nhiên là chứa source code rồi... Đặc biệt quan tâm
  • package.json: Chứa thông tin phiên bản cài đặt của các package npm... Quan tâm tí tẹo thôi
  • Các file còn lại… kệ nó đi
